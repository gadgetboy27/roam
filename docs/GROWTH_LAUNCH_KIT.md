# roam. — Growth Launch Kit (2026-06-13)

The product is ready: free core loop (match · message · connect, all free + unlimited),
fast, no fake profiles, photo-first onboarding, identity verification, referral loop,
public event RSVP, social sharing, and a **live founding-member counter** (46/50 spots
left → urgency). The bottleneck now is **distribution** — real-world action only Henry
can take. This kit makes that execution concrete.

## The strategy in one line
**Win one city × one niche first.** A dense 50-person Auckland (or Queenstown) hiking
crowd beats 500 scattered users — because density is what makes the swipe feed and
events feel alive. Don't go wide yet.

## The loops that are built and waiting for traffic
- **Founding scarcity** → landing shows "🔥 Only N of 50 spots left — free Adventurer
  for life." Converts visitors.
- **Referral** → every user's Home has "Invite your crew"; invited friends auto-connect.
- **Events** → create a real event, hit Share → `letsroam.life/e/<id>`; strangers RSVP +
  auto-join. The single best acquisition wedge (joining the hike requires the app).
- **Social share sheet** → native + Facebook/X/WhatsApp/Telegram/email everywhere.

## Week-1 action checklist (Henry)
1. **Pick the beachhead:** Auckland + tramping/hiking (or Queenstown). Commit to one.
2. **Seed supply (where adventurers already are):** post the copy below in 5 NZ outdoor
   communities — tramping/hiking & climbing FB groups, uni outdoor clubs, climbing-gym
   noticeboards, local Meetup groups, a run/surf club.
3. **Host ONE real meetup** (a Sunday hike). Create it in-app → Share the `/e/<id>` link
   in those communities. Every RSVP = a real, local, activity-matched signup.
4. **Personal invites:** send the invite link to 10–20 adventurous friends (use Home →
   Invite your crew). They arrive already connected to you = feed isn't empty.
5. **Line up 1 organiser** (a local hike/club leader) to run their group on roam — brings
   their existing crowd (the Meetup model; this is also your Squad-Leader revenue).
6. **Watch the funnel:** signups → profile complete (photo) → first swipe → first match.

## What converts better (already built, just use it)
- Lead with **"free for life, 46 spots left"** — scarcity + free is the strongest hook
  at this stage.
- Always pair a signup ask with a **real event** — "join + come on the hike" out-converts
  "sign up."
- Get every new user to **add 1 photo** (onboarding nudges this) — blank profiles don't
  match.
